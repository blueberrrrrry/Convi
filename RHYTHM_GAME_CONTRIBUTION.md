# 3D 리듬 액션 게임 - 역할과 기여

## 역할과 기여
- **프론트엔드 웹 개발**: React 18 + TypeScript 기반 게임 홈페이지 개발, 네온 효과 랜딩 페이지 구현, 스크롤 애니메이션 및 반응형 레이아웃 적용, React Router v6를 활용한 라우팅 구조 설계
- **Unity 게임 엔진 개발**: Unity C# 기반 3D 게임 로비 씬 구현, 학교 환경 3D 씬 제작 및 최적화, SchoolLobbyManager를 통한 씬 전환 및 UI 관리 시스템 구현
- **리듬 게임 코어 시스템**: 마우스 클릭 기반 4레인 노트 처리 로직 구현, 음악 BPM 동기화 노트 생성 시스템 개발, Perfect/Great/Good/Miss 4단계 판정 시스템 구현
- **게임 점수 시스템**: 판정별 차등 점수 계산 로직 구현, 콤보 배율 시스템 개발, 최종 점수 및 랭크 표시 화면 구현
- **3D 캐릭터 애니메이션**: MMD4Mecanim을 활용한 3D 캐릭터 애니메이션 연동, 리듬에 맞춘 캐릭터 동작 동기화 구현
- **백엔드 및 데이터베이스**: Supabase를 활용한 사용자 데이터 및 점수 기록 저장 시스템 구현, PostgreSQL 데이터베이스 스키마 설계, RESTful API를 통한 게임 데이터 연동
- **전체 시스템 통합**: 웹 홈페이지와 Unity 게임 간 연동 구조 설계, 게임 시작 CTA 버튼을 통한 웹-게임 전환 플로우 구현, Vite 빌드 시스템을 활용한 프로덕션 배포 최적화
